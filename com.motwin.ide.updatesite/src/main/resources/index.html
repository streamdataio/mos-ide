Motwin IDE README

Server IDE Installation

  1. Download Eclipse 3.8 from http://download.eclipse.org/eclipse/downloads/eclipse3x.html;
  2. Install and Start Eclipse
  3. Install Spring Tools Suite:
    1. Click "Help" > "Install New Software...";
    2. Click "Add...";
    3. Enter http://dist.springsource.com/release/TOOLS/update/e3.8 in "Location";
    4. Select "Core / Spring IDE";
    5. Click "Finish" and accept licences agreements;
    6. Restart Eclipse.
  4. Install Virgo Tooling:
    1. Click "Help" > "Install New Software...";
    2. Click "Add...";
    3. Enter http://download.eclipse.org/virgo/release/tooling in "Location";
    4. Select "Eclipse Virgo Tools";
    5. Click "Finish" and accept licences agreements;
    6. Restart Eclipse.
  5. Install Motwin IDE for Server SDK:
    1. Click "Help" > "Install New Software...";
    2. Click "Add...";
    3. Enter https://new-central.mootwin.net/sdk/IDE/SNAPSHOT/ in "Location";
    4. Select "Motwin IDE for Server SDK";
    5. Click "Finish" and accept licences agreements;
    6. Restart Eclipse.

Android IDE Installation

  1. Download Eclipse 3.8 from http://download.eclipse.org/eclipse/downloads/eclipse3x.html;
  2. Install and Start Eclipse
  3. Install Android Eclipse ADT:
    1. Click "Help" > "Install New Software...";
    2. Click "Add...";
    3. Enter http://dl.google.com/android/eclipse/ in "Location";
    4. Select "Developer Tools";
    5. Click "Finish" and accept licences agreements;
    6. Restart Eclipse.
  4. Install Motwin IDE for Android SDK:
    1. Click "Help" > "Install New Software...";
    2. Click "Add...";
    3. Enter https://new-central.mootwin.net/sdk/IDE/SNAPSHOT/ in "Location";
    4. Select "Motwin IDE for Android SDK";
    5. Click "Finish" and accept licences agreements;
    6. Restart Eclipse.
