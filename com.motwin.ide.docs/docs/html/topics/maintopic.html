Motwin Platform

Motwin offers enterprises and developers a middleware platform to build and operate connected mobile and Web applications.

With Motwin Platform developers can:

  1. Build High Performance Applications by optimizing the interactions with back-ends and 3rd party providers through API orchestration, cache and data aggregation.
  2. Enable Real-Time Applications with live streaming and network resilience capabilities.
  3. Create Highly Personalized Content using Motwin context-aware SDKs, in-app notifications and connectors to better engage and serve their customers.
  4. Shorten the Development Phase and Better Manage the Application Life Cycle thanks to Motwin's unique implementation model that fully decouples business and presentation logics.

With Motwin Platform, developers can create sophisticated native, HTML5, hybrid and Web applications. Enterprises can accelerate their time to market, implement a true multi-channel strategy and deliver value-added services to their customers and employees.

Develop hyper-responsive applications thanks to Motwin network resilient capabilities built on top of its patented technology.

Motwin Platform provides unique context management features for developers to create stunning user experiences and create engaging and personalized contents. Our high performance capabilities combined with optimized data transfer make it easy for enterprises to develop and deploy efficient mobile applications.

Features

Motwin Platform is a powerful environment for developers to build sophisticated and responsive applications. It offers key technical functionalities that address the typical challenges encountered when building applications. With Motwin Platform developers can focus on their specific business logic and let Motwin deal with the technical complexity of connected and high-performance applications.

Motwin Platform includes the following components:

  • Motwin mOS (Motwin Operation Server): to run server-side apps.
  • Motwin Development Environment: to easily create, test and deploy Motwin applications.
  • Motwin Server SDK: to build the server-side business logic of your apps, optimize your integration, and process mobile and server contexts.
  • Motwin Connectors and Connector API: to efficiently, securely and quickly retrieve, update and aggregate data from existing enterprise and cloud services.
  • Motwin Technical Modules: to provide shared functionality to Motwin apps (e.g.: user and device management, rich-content messaging, unified and in-app notifications, etc.)
  • Motwin Web Portal: to provide administration of the Motwin Platform and your apps on a dashboard.
  • Motwin iOS SDK: to process mobile context and data for iOS mobile platforms - this SDK integrates with Xcode.
  • Motwin Android SDK: to process mobile context and data for Android mobile platforms - this SDK integrates with Eclipse.
  • Motwin HTML5 SDK: to process mobile context and data transfer for HTML 5 applications, mobile or Web.

Key functionality:

  • API management: offers orchestration, data aggregation, translation and optimization
  • Caching and Incremental data transfer: helps minimize the volume of data transferred during communications.
  • Real-time APIs: provides developers with a simple set of APIs for real-time data processing
  • Network connectivity management: addresses network connectivity problems for developers.
  • Context Management: simplifies the real-time collection and processing of user contexts (mobile, Web and server side).

Motwin Operation Server (mOS) runs server-side apps and manages the communications between clients and back-ends, cloud services, and third party providers. It is built to host high-performance applications.

Motwin server-side SDK is used to build the shared business logic for multiple clients (iOS, Android or HTML5 - mobile or Web.)

Mobile (iOS and Android) and HTML5 SDKs permit developers to create optimal applications leveraging the server-side business logic and focusing primarily on a rich and engaging user experience.

Motwin Architecture

Mobile is triggering changes in how applications are consumed and delivered. Transitioning to a more flexible application delivery architecture is critical for enterprises that want to stay in the competition. A middleware platform that enables an efficient API management while decoupling back-ends (DB, Cloud Service, Legacy systems) from mobile applications is critical to deliver modern and efficient applications. Motwin server provides all the key infrastructure support services for the management, delivery, and security of APIs and enables enterprises to deliver business services rapidly, securely and consistently.

Motwin architecture makes development of applications for different mobile operating systems highly efficient and powerful. The code developed on the server is used by each mobile application, minimizing the amount of code required for each mobile platform you need to develop and maintain.

The runtime Motwin server sits between mobile devices and the enterprise information systems (Legacy systems, Cloud services) and acts as a mediator in charge of fetching and aggregating data from various sources to bridge the gap between legacy infrastructure and the fast changing world of Cloud and mobile.

Motwin provides mobile SDKs for each of today's major mobile operating systemts to allow developers to consistently access their back-end services in a unified manner regardless of the type of applications they are building (native, Web or Hybrid).
