Motwin Developer Guide

This document describes developing server-side applications to take advantage of the optimizations offered by the Motwin Platform. It introduces important development concepts and provides examples of typical application tasks. It also describes client-side app development for Android, iOS, and HTML5 applications. It contains instructions on setting up the developer environment for each of these platforms.

Preface

For enterprises and developers, writing mobile applications can be cumbersome and time consuming. The Motwin Platform provides a set of technologies and tools designed to solve common problems related to the development of data-rich enterprise and business applications. Mobile applications that have stringent requirements in terms of performance, real-time data exchange, or that must work with degraded network connections will benefit the most from the Motwin Platform.

This document describes the server-side and client-side development processes for creating applications on the Motwin Platform and provides information about configuring and managing the server-side system.
